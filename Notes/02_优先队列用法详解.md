## 2. 优先队列简介(priority_queue)

- 普通的队列是一种先进先出的数据结构，元素在队列尾追加，而从队列头删除。  

- 在优先队列中，元素被赋予优先级。当访问元素时，具有最高优先级的元素最先删除。优先队列具有最高级先出 （first in, largest out）的行为特征。



## 3. 常用接口

- top 访问队头元素
- empty 队列是否为空
- size 返回队列内元素个数
- push 插入元素到队尾 (并排序)
- emplace 原地构造一个元素并插入队列
- pop 弹出队头元素
- swap 交换内容



## 4. 自定义数据类型排序样例

直接放代码，只可意会不可言传

```C++
struct teacher
{
    int w, s, c;
    bool operator<(const teacher &a) const //运算符重载改变排序方式，从小到大排列
    {
        if (a.s == s)
        {
            return a.c < c;
        }
        else
        {
            return a.s < s;
        }
    }
};
```
